The flag resolution modal has two editable inputs — Ground Truth Label and Attributes — and one auto-derived output — Reviewer Assessment.

How it works:

  1. Leave both untouched → assessment locks to Accepted
  2. Change the label only → Label Corrected
  3. Toggle an attribute chip only → Attributes Corrected
  4. Change both → Label & Attributes Corrected

The highlighted row under Reviewer Assessment updates in real time as you edit. Add a note back to the annotator, click Resolve, and the flag moves to the Resolved list — where the diff you just made is preserved.

Every resolve captures a before / after snapshot so the record stays auditable — even if the underlying image gets re-edited later.

How the diff renders:

  1. Label change → PREV → NEW
  2. Attribute added → green chip with + prefix
  3. Attribute removed → red strikethrough chip with − prefix
  4. Attribute kept → neutral sky chip

Every control is read-only in this view — reviewers can't accidentally re-edit an already-resolved flag.
